## 插件分享  
  > 插件路径打包分享:`C:\Users\你的用户名\.vscode\extensions`

## 排除对应类型文件的展示

> setting.json
```json
"files.exclude": {
    "**/.git": true,
    "**/.svn": true,
    "**/.hg": true,
    "**/CVS": true,
    "**/.DS_Store": true,
    "**/*.spec.ts": true
}
```
